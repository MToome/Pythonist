##### Boolean 
Boolean on väljend, on kaks varjanti kas tõene(True) või vale(False). kui näiteks võrdled kahte väärtust siis pyton hindab seda ja annab Boolean vastuse.
```python
print(1 < 11)
# Väljund True
print(32 < 1)
# Väljund False
```
